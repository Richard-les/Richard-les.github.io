js中的对象、函数、构造函数、实例对象以及函数与对象的关系

对象的概念就是对象就是一组“键值对”（key-value）的集合，是一种无序的复合数据集合。在代码中，每当声明变量后的大括号里就定义了一个对象，它被赋值给了声明的变量，所以该变量就指向了一个对象，对象内部可以继续添加键值对，前面的叫键名，冒号后的为键值。键名可以是数值也可以是字母，如果键名不符合标识名的条件（比如第一个字符为数字，或者含有空格或运算符），且也不是数字，则必须加上引号，否则会报错。对象每一个键名称为属性，如果属性的值为函数，就称属性为方法。属性可以动态创建不必在对象声明时就指定。在对象中还有一个比较特殊的存在叫实例对象，对象有专属于自己的属性，还有一个[prototype]指向可以通过prototype访问；。

我觉得在说对象方法前，有必要先了解一下函数的定义。函数是一段可以反复调用的代码块。函数还能接受输入的参数，不同的参数会返回不同的值。函数的声明总共有三种方法，第一种是function命令，function后是函数名，函数名后面是一对圆括号，里面是传入函数的参数。函数体放在大括号里面。第二种就是使用函数表达式，将一个匿名函数赋值给变量。这时，这个匿名函数又称函数表达式，因为赋值语句的等号右侧只能放表达式。采用函数表达式声明函数时，function命令后面不带有函数名。如果加上函数名，该函数名只在函数体内部有效，在函数体外部无效。第三种就是使用function构造函数，你可以传递任意数量的参数给Function构造函数，只有最后一个参数会被当做函数体，如果只有一个参数，该参数就是函数体。但这种方法不够直观也比较麻烦，所以不常使用。调用函数时，要使用圆括号运算符。圆括号之中，可以加入函数的参数。函数体内部的return语句，表示返回。JavaScript 引擎遇到return语句，就直接返回return后面的那个表达式的值，后面即使还有语句，也不会得到执行。也就是说，return语句所带的那个表达式，就是函数的返回值。return语句不是必需的，如果没有的话，该函数就不返回任何值，或者说返回undefined。

下来就说说对象方法，JavaScript 方法是能够在对象上执行的动作，JavaScript 方法是包含函数定义的属性。在 JavaScript 中，被称为 this 的事物，指的是拥有该 JavaScript 代码的对象，this 的值，在函数中使用时，是“拥有”该函数的对象，请注意 this 并非变量，它是关键词，无法改变 this 的值。

说到函数与对象的关系，我认为有必要先知道，在js中，一切都是对象，函数也是对象，对象都是由函数创建的。
